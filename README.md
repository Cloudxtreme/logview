logview
=======

This program is a very simple logfile colorizer. It removes a few
columns (month, day, hostname, ...) from original file to make it more
readable.

Run "./logview -h" for usage.

It requires following Perl modules:
 File::Tail
 Term::ANSIColor
 Term::ReadKey
